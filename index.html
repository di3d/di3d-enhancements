di3d's Enhancements

Minecraft 26.1.2  ·  Client-side  ·  Works with vanilla servers

1

Download Prism Launcher

Prism is a free launcher that makes installing modpacks simple.

Get Prism Launcher ↗
On Windows, grab the installer (.exe). Run it, click through — defaults are fine.
2

Download the modpack

Download the .mrpack file and save it somewhere you can find it.

Download di3d's Enhancements ↓
3

Add a new instance in Prism

Open Prism Launcher and click Add Instance (top-left).

In the dialog, select Import from the left sidebar, then click Browse and choose the .mrpack file you just downloaded. Click OK.
4

Log in and launch

Click the instance, then hit Launch. Prism will prompt you to log in with your Microsoft account if you haven't already.

You only need to log in once. The mods are client-side — you can join any vanilla 26.1.2 server as usual.

Something not working? The pack requires Java 21. Prism can install it for you: Settings → Java → Auto-detect or download from Adoptium.
